Primeiro trabalho em C++

Explicação do código utilizado para realizar o trabalho

#include iostream

#include map

#include string

Usando o namespace std;

Nesta parte do código, estamos incluindo as bibliotecas iostream, map e string. A biblioteca iostream é usada para entrada e saída padrão. A biblioteca map é uma estrutura de dados que associa chaves a valores e é usada para manipular strings. A declaração 'using namespace std;' nos permite usar elementos do namespace std sem precisar prefixá-los com std::


struct Material

Estamos definindo uma estrutura chamada Material, que possui quatro membros: nome (uma string que armazena o nome do material), estoque (um inteiro que armazena a quantidade em estoque), preco (um número de ponto flutuante que armazena o preço do material) e vendidos (um inteiro que armazena a quantidade vendida do material).


int main()

A função main() é o ponto de entrada do programa. Aqui, estamos declarando uma variável materiais como um mapa onde as chaves são inteiros e os valores são do tipo Material. Também estamos inicializando uma variável lucroTotal como um número de ponto flutuante que armazenará o lucro total obtido com as vendas.


Inicializando os materiais

Aqui, estamos inicializando os materiais dentro do mapa materiais, atribuindo valores iniciais para cada material.


Menu de Opções

Nesta parte do código, estamos usando um loop while (true) para permitir que o usuário faça repetidas escolhas de opções até que ele escolha sair do programa. Dentro do loop, estamos exibindo um menu com cinco opções usando cout, e pedindo ao usuário que escolha uma opção digitando um número, que é armazenado na variável opcao.


Opção 1: Vender produto

Se o usuário escolher a opção 1, o programa permite vender um produto. Ele exibe os materiais disponíveis com seus códigos, estoques, preços e quantidade vendida. O usuário então digita o código do material que deseja vender e a quantidade. Se o código for válido e houver estoque suficiente, a quantidade vendida é subtraída do estoque, o lucro total é atualizado e uma mensagem de venda bem-sucedida é exibida. Caso contrário, uma mensagem de erro é mostrada.


Opção 2: Mostrar estoque

Se o usuário escolher a opção 2, o programa exibirá o estoque atual de cada material, mostrando o nome, quantidade em estoque e preço.


Opção 3: Mostrar lucro total

Se o usuário escolher a opção 3, o programa exibirá o lucro total obtido com todas as vendas realizadas até o momento.


Opção 4: Comprar produto para repor estoque

Se o usuário escolher a opção 4, o programa permitirá comprar produtos para repor o estoque. Ele solicitará o código do material e a quantidade a ser comprada. Após a compra, o estoque será atualizado e uma mensagem de confirmação será exibida.


Opção 5: Sair do programa

Se o usuário escolher a opção 5, o programa exibirá uma mensagem de saída e sairá do loop usando a instrução 'break'.
